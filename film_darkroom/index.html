黑白空间 · B&W Space 隐私政策

最后更新日期：2026年7月7日

重要提示：本应用是一款完全离线运行的黑白胶片冲洗工具。所有数据均存储在您的设备本地，我们不会收集、传输或存储您的任何个人信息到远程服务器。

一、概述

黑白空间 · B&W Space（以下简称「本应用」）是一款帮助黑白胶片摄影者在冲洗时快速查询显影参数并用倒计时执行冲洗流程的离线工具。它内置完整的黑白显影数据库，按 厂商 → 型号 → 药水 定位冲洗参数；提供显影倒计时（含多阶段冲洗流程、搅拌提醒、温度补偿）；支持您自录配方、收藏、冲洗历史，以及配方的 JSON 导入导出。我们非常重视您的隐私保护和个人信息安全。本隐私政策旨在向您说明我们如何处理和保护您的信息。

使用本应用即表示您已阅读、理解并同意接受本隐私政策的全部内容。如果您不同意本隐私政策的任何内容，请立即停止使用本应用。

二、信息收集

2.1 我们不收集的信息

本应用不会收集以下任何个人信息：

  • 姓名、电子邮件地址、电话号码等身份识别信息（应用无需注册登录）
  • 位置信息（GPS 或网络定位）
  • 设备唯一广告标识符（如 IDFA 等）
  • 联系人信息、通话记录、短信内容
  • 生物识别信息（指纹、面容 ID 等）
  • 网络浏览历史
  • 任何形式的可识别个人身份的信息

2.2 本地存储的数据

为了提供冲洗查询与倒计时功能，本应用会在您的设备本地存储以下数据：

  • 用户配方：您自行录入的显影配方（厂商、型号、药水、稀释比例、ISO、各画幅显影时间、温度、备注等），以及您从内置记录「编辑另存为副本」派生出的配方
  • 收藏记录：您标星的配方（可跨内置与用户记录）及其收藏时间
  • 冲洗历史：完成（或手动结束）倒计时时写入的参数快照，包括显影药水、温度、实际用时、是否使用了温度补偿、起止时间等
  • 偏好设置：主题（浅色 / 深色 / 暗房）、语言、搅拌提醒开关与间隔、默认温度、冲洗流程默认值、自定义温度补偿曲线等个性化设置

重要说明：以上数据仅存储在您的设备本地（基于本地 SQLCipher 加密数据库与系统偏好存储），不会上传到任何服务器，不会与我们或任何第三方共享。

需要特别说明的是，应用内置的显影数据库属于应用自带的参考资料，在本地以加密形式存放、仅载入内存供查询，不会被批量导出，也不包含任何与您个人身份相关的信息。

三、权限使用说明

本应用仅在您主动授权的情况下访问以下设备功能：

3.1 通知权限

使用目的：

  • 在冲洗倒计时到达终点时发送本地通知（响铃 + 震动），即便应用切到后台、屏幕锁定或被系统回收，也能到点提醒
  • 在多阶段冲洗流程中，于每个阶段结束时提醒您进入下一步

使用方式：仅在您启动倒计时时，由系统在本地按预约的完成时刻发送通知，不经过任何远程推送服务器。

拒绝影响：如果您拒绝此权限，应用切到后台或锁屏后将无法到点提醒，但倒计时在应用前台时仍可正常运行。

3.2 音频播放权限

使用目的：

  • 播放冲洗倒计时完成时的提示铃声
  • 播放搅拌提醒的短促提示音

使用方式：仅在倒计时进行或到达终点时播放提示音；音频类别配置为警报播放，确保在设备静音模式下也能发出提示（暗房环境下您不会错过关键节点）。

拒绝影响：如果您拒绝此权限或关闭响铃，您将听不到完成与搅拌提示音（但仍会收到通知与震动）。

3.3 文件访问权限

使用目的：

  • 将您自己录入或派生的配方导出为 JSON 文件进行备份或迁移
  • 从 JSON 文件导入配方到本应用

使用方式：仅在您主动点击「导出」或「导入」功能时才会访问文件，且导出内容仅限您自己的用户配方（内置数据库不可导出）。

拒绝影响：如果您拒绝此权限，您将无法导出或导入配方文件，但应用内的所有浏览、查询、倒计时与编辑功能仍可正常使用。

3.4 其他权限

本应用不会请求以下权限：

  • 相机权限
  • 麦克风权限
  • 位置权限
  • 通讯录权限
  • 相册 / 照片库权限
  • 任何社交账号或网络账号权限

四、数据使用和共享

4.1 数据使用

我们使用本地存储的数据仅用于：

  • 记录和展示您的配方、收藏与冲洗历史
  • 执行显影倒计时、多阶段流程计时、搅拌提醒与温度补偿
  • 在本地生成冲洗相关的统计与进度展示
  • 执行您主动发起的配方导入 / 导出
  • 保存您的应用偏好设置（主题、语言、搅拌提醒、默认参数等）

4.2 数据共享

我们不会与任何第三方共享您的数据，具体包括：

  • 不出售、出租或交易您的任何信息
  • 不向广告商、数据分析服务商提供任何数据
  • 不与社交媒体平台共享任何信息
  • 不与任何其他第三方公司或组织共享您的数据

4.3 数据导出和分享

本应用提供配方导出功能，允许您：

  • 将您自己录入或从内置记录派生的用户配方导出为 JSON 文件进行备份
  • 通过系统分享功能将导出文件发送到您指定的位置

这些操作完全由您主动触发，导出范围严格限于用户配方。我们不会在后台自动上传或分享您的任何数据；内置显影数据库受应用内加密保护，不可被批量导出。

五、第三方服务

5.1 使用的第三方组件

本应用使用了以下开源组件来提供功能：

  • Flutter SDK：应用开发框架
  • Riverpod：状态管理库
  • Drift & SQLite（SQLCipher）：本地加密数据库
  • flutter_secure_storage：通过 iOS Keychain 安全存储数据库密钥
  • cryptography：用于内置数据的本地 AES 加密保护
  • flutter_local_notifications：本地通知库
  • just_audio / audio_session：音频播放与会话配置
  • wakelock_plus：冲洗期间保持屏幕常亮
  • share_plus：系统分享功能
  • file_picker：文件选择器（配方导入）
  • path_provider / uuid / intl / shared_preferences：本地路径、标识生成、格式化与偏好存储
  • 其他 UI 组件库：用于界面展示

以上组件均为纯本地功能库，不涉及将您的数据发送到任何远程服务器。

5.2 不使用的第三方服务

本应用不使用以下任何服务：

  • 用户行为分析服务（如 Google Analytics、Firebase Analytics 等）
  • 广告服务或广告联盟
  • 社交登录服务（如微信登录、Apple 登录等）
  • 云存储与云同步服务（如 iCloud、Google Drive 等）
  • 远程推送通知服务（仅使用 iOS 本地通知）
  • 崩溃报告与远程日志服务
  • 客户支持或反馈追踪工具

六、数据安全

本应用所有数据均存储在您的设备本地，并采取以下措施保护您的信息安全：

  • 本地沙盒：利用 iOS 系统级沙盒机制保护应用数据，其他应用无法读取
  • 加密数据库：您录入的配方、收藏与冲洗历史存储在 SQLCipher 加密的本地数据库中，数据库密钥保存在 iOS Keychain 内，离开您的设备无法解密
  • 内置数据加密：应用自带的显影数据库以 AES 加密形式打包，仅在运行时解密载入内存，明文从不写入磁盘
  • 容器锁定：应用已关闭文件共享与文档原地打开，应用文件在访达 / 文件 App 中不可见，也不计入 iTunes / 访达的文件共享备份
  • 无网络传输：应用无任何网络请求，不存在数据在传输过程中被截获的风险
  • 导出隔离：导出路径在架构上无法触达内置数据库，仅能导出您自己的用户配方

请注意，如果您丢失设备或设备被盗，设备上的本地数据可能被他人访问。建议您启用设备的密码锁或面容 ID / 指纹识别功能，并定期使用导出功能将重要配方备份到安全位置。

需要说明的是，本地保护无法绝对防御越狱设备上的恶意逆向行为，但足以保证普通使用场景下您的数据不会被随意获取。

七、数据保留和删除

7.1 数据保留期限

由于所有数据均存储在您的设备本地，数据保留期限完全由您决定。只要您不主动删除，数据将一直保留在您的设备上。

7.2 数据删除

您可以通过以下方式删除数据：

  • 删除单条记录：在应用内直接删除不需要的配方、收藏或历史记录
  • 清除所有数据：在应用设置中选择清除全部数据（建议先导出备份）
  • 卸载应用：卸载本应用将删除所有应用数据（建议先导出备份）

数据被删除后无法恢复，请谨慎操作。建议在删除前使用导出功能保存重要配方。

八、儿童隐私保护

本应用适合所有年龄段的用户使用。由于本应用不收集任何个人信息，不进行用户注册，不进行网络数据传输，因此特别适合未成年人使用。

家长和监护人可以放心让儿童使用本应用，我们不会知晓或收集儿童的任何个人信息。

建议您定期查阅本隐私政策，以了解我们如何保护您的信息。
