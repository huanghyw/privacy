存钱管家 · Money Saving Manager 隐私政策

最后更新日期：2026年6月24日

重要提示：本应用是一款完全离线运行的个人储蓄追踪应用。所有数据均存储在您的设备本地，我们不会收集、传输或存储您的任何个人信息到远程服务器。

一、概述

存钱管家 · Money Saving Manager（以下简称「本应用」）是一款帮助用户通过结构化储蓄计划（365 天递增、52 周挑战、12 个月固定、每日固定、自由存等）养成存钱习惯的个人理财工具，配合每日打卡、连击激励和数据报表，让储蓄过程可视化、有成就感。我们非常重视您的隐私保护和个人信息安全，尤其是涉及财务的数据。本隐私政策旨在向您说明我们如何处理和保护您的信息。

使用本应用即表示您已阅读、理解并同意接受本隐私政策的全部内容。如果您不同意本隐私政策的任何内容，请立即停止使用本应用。

二、信息收集

2.1 我们不收集的信息

本应用不会收集以下任何个人信息：

  • 姓名、电子邮件地址、电话号码等身份识别信息（应用无需注册登录）
  • 位置信息（GPS 或网络定位）
  • 设备唯一广告标识符（如 IDFA 等）
  • 联系人信息、通话记录、短信内容
  • 生物识别信息（指纹、面容 ID 等）
  • 银行账户、支付密码、卡片信息等真实金融账户凭证
  • 网络浏览历史
  • 任何形式的可识别个人身份的信息

2.2 本地存储的数据

为了提供储蓄追踪与打卡激励功能，本应用会在您的设备本地存储以下数据：

  • 用户资料：您自定义的昵称、头像（仅本地展示，无需注册）
  • 储蓄计划：计划类型、目标金额、起止时间、进度与状态
  • 存入记录：每次存入的金额、日期及相关备注
  • 打卡记录：每日签到、每周签到记录及连续打卡天数（连击）
  • 报表数据：由上述记录在本地汇总生成的趋势、分布、热力图等统计结果
  • 偏好设置：主题配色、深色模式、货币显示格式、提醒时间等个性化设置
  • 备份文件：您主动导出的加密备份及其元信息

重要说明：以上数据仅存储在您的设备本地（基于本机 Hive 数据库），不会上传到任何服务器，不会与我们或任何第三方共享。

需要特别说明的是，本应用记录的储蓄金额仅为您自行输入的记账数字，用于个人存钱进度管理，应用本身不会连接任何银行、支付或金融机构，也不会读取您的真实账户余额。

三、权限使用说明

本应用仅在您主动授权的情况下访问以下设备功能：

3.1 通知权限

使用目的：

  • 发送每日 / 每周签到提醒，帮助您坚持存钱打卡
  • 根据您配置的时间点（TimeOfDay）准时推送本地提醒

使用方式：仅在您开启了提醒并设置了提醒时间后，由系统在本地发送通知，不经过任何远程推送服务器。

拒绝影响：如果您拒绝此权限，您将不会收到签到提醒，但所有储蓄与打卡功能仍可正常使用。

3.2 文件访问权限

使用目的：

  • 将您的数据导出为 AES 加密备份文件，便于保存或迁移到新设备
  • 导入此前导出的加密备份以恢复数据

使用方式：仅在您主动点击「导出备份」或「导入备份」时才会访问文件，且备份文件由您设置的密码进行 AES 加密保护。

拒绝影响：如果您拒绝此权限，您将无法导出或导入备份，但应用内的所有功能仍可正常使用。

3.3 其他权限

本应用不会请求以下权限：

  • 相机权限
  • 麦克风权限
  • 位置权限
  • 通讯录权限
  • 任何社交账号或网络账号权限

四、数据使用和共享

4.1 数据使用

我们使用本地存储的数据仅用于：

  • 记录和展示您的储蓄计划与存入进度
  • 统计连续打卡天数、计划完成度等激励信息
  • 在本地生成趋势、分布、热力图等可视化报表
  • 执行您主动发起的加密备份与恢复
  • 保存您的应用偏好设置（主题、货币格式、提醒时间等）

4.2 数据共享

我们不会与任何第三方共享您的数据，具体包括：

  • 不出售、出租或交易您的任何信息
  • 不向广告商、数据分析服务商提供任何数据
  • 不与社交媒体平台共享任何信息
  • 不与银行、支付机构或任何金融机构共享任何信息
  • 不与任何其他第三方公司或组织共享您的数据

4.3 数据导出和分享

本应用提供加密导出功能，允许您：

  • 将储蓄与打卡数据导出为 AES 加密备份文件进行保存
  • 通过系统分享功能（share_plus）将备份文件发送到您指定的位置

这些操作完全由您主动触发，并由您设置的密码加密保护。我们不会在后台自动上传或分享您的任何数据。

五、第三方服务

5.1 使用的第三方组件

本应用使用了以下开源组件来提供功能：

  • Flutter SDK：应用开发框架
  • Provider：状态管理库
  • Hive / hive_flutter：本地 NoSQL 数据库
  • fl_chart：图表可视化库
  • Lottie：动画渲染库
  • google_fonts：字体渲染库
  • crypto / encrypt：用于本地 AES 加密备份
  • path_provider：获取本地文件存储路径
  • share_plus：调用系统分享功能
  • device_info_plus：读取本地设备信息（仅用于备份文件标识等本地功能，不会上传）
  • shared_preferences / uuid / intl / logging：偏好存储、标识生成、格式化与本地日志

以上组件均为纯本地功能库，不涉及将您的数据发送到任何远程服务器。

5.2 不使用的第三方服务

本应用不使用以下任何服务：

  • 用户行为分析服务（如 Google Analytics、Firebase Analytics 等）
  • 广告服务或广告联盟
  • 社交登录服务（如微信登录、Apple 登录等）
  • 云存储与云同步服务（如 iCloud、Google Drive 等）
  • 远程推送通知服务（仅使用 iOS 本地通知）
  • 崩溃报告与远程日志服务
  • 客户支持或反馈追踪工具

六、数据安全

虽然本应用所有数据均存储在您的设备本地，我们仍然采取以下措施保护您的信息安全：

  • 本地沙盒：利用 iOS 系统级沙盒机制保护应用数据，其他应用无法读取
  • 加密备份：导出的备份文件采用 AES 加密，由您设置的密码保护，无密码无法解密
  • 自动本地备份：应用可在本地定时生成备份，帮助您防范数据丢失
  • 无网络传输：不存在数据在传输过程中被截获的风险
  • 最小化原则：仅记录与储蓄追踪功能相关的必要数据

请注意，如果您丢失设备或设备被盗，设备上的本地数据可能被他人访问。建议您启用设备的密码锁或面容 ID / 指纹识别功能，并定期使用加密导出功能将备份保存到安全位置。

七、数据保留和删除

7.1 数据保留期限

由于所有数据均存储在您的设备本地，数据保留期限完全由您决定。只要您不主动删除，数据将一直保留在您的设备上。

7.2 数据删除

您可以通过以下方式删除数据：

  • 删除单条记录：在应用内删除不需要的存入记录或签到记录
  • 删除储蓄计划：在计划卡片上滑动或进入详情删除整个计划
  • 清除所有数据：在应用设置中选择清除全部数据（建议先导出备份）
  • 卸载应用：卸载本应用将删除所有应用数据（建议先导出备份）

数据被删除后无法恢复，请谨慎操作。建议在删除前使用加密导出功能保存重要数据。

八、儿童隐私保护

本应用适合所有年龄段的用户使用。由于本应用不收集任何个人信息，不进行用户注册，不进行网络数据传输，因此特别适合未成年人使用。

家长和监护人可以放心让儿童使用本应用，我们不会知晓或收集儿童的任何个人信息。

九、敏感信息说明

本应用涉及个人储蓄金额等具有财务属性的数据，请注意：

  • 所有储蓄金额、打卡记录仅存储在您的设备本地，不会传输到任何服务器
  • 应用记录的金额仅为您自行输入的记账数字，不连接任何真实金融账户，也不会读取您的银行余额或交易明细
  • 导出的备份文件由您设置的密码进行 AES 加密，请妥善保管密码，丢失密码将无法恢复备份
  • 本应用提供的报表与统计仅用于个人储蓄管理参考，不构成任何投资或理财建议
  • 您可以随时删除任何记录或全部数据
